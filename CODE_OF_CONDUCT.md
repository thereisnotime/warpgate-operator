# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants are expected to uphold this code. Please report unacceptable behavior via
[GitHub Security Advisories](https://github.com/thereisnotime/warpgate-operator/security/advisories/new)
or by contacting the maintainers directly.
